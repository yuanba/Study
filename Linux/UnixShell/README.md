# Unix / Linux Shell
    本笔记针对与书目<<精通Unix Shell脚本编程>>，使用shell脚本学习解决现实生活中的Linux或者Unix任务
    **永远不止一种解决问题的方法，至少在Shell中是这样**
    有以下的细节在学习的时候需要注意：
    * 每一步的注释
    * 合成命令组合成单一命令提高效率
    * 有时候需要将命令分割以便于理解
    * 变量和文件存储在长周期使用shell脚本中又很大的优势
    * 因为大写字母通常代表系统中的环境变量，所以我们在自己编写的shell程序中需要小心使用这些大写单词的变量名称

    **本书同样适用于所有的linux发行版**
